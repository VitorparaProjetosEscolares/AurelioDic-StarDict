<h2>Como instalar o dicionário no seu KoReader</h2>

1. Primeiro baixe o arquivo dentro do releases e extraia com o WinRAR ou 7-Zip;
2. Então, acesse a pasta <code>koreader/data/dict/</code> e coloque a pasta já extraída dentro desta;
3. Reinicie o KoReader e voilà! Agora você tem o dicionário do Marco Aurélio para entender aquelas palavras complicadíssimas.
